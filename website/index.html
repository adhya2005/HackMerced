Welcome to Crop Finder

A website user interface that predicts specific crop yields in certain locations based on a weather app API and soil type. The user interface will consist of a typing menu where you will input location, Nitrogen, Phosphorous, Potassium, and Temperature (decided through a weather API), humidity, soil pH value, and rainfall in mm. When a location has been selected, an image of a zoomable map of that area will appear with whatever choices selected and a prediction of that information. The predict button will output if that crop is likely to have a good yield in that environment.
